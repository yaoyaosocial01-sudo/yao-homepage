# YAO / 瑶瑶个人主页

这是 YAO / 瑶瑶的个人品牌主页项目，用来展示她的生物医药背景、澳洲工作生活、内容创作、播客、数字分身入口和合作联系方式。

当前版本是准备上线版。项目使用普通 HTML、CSS 和 JavaScript，没有复杂框架，所以可以直接本地打开，也适合之后部署到 Vercel 或 Netlify。

## 本地怎么打开

直接双击打开这个文件：

```text
D:\个人主页\index.html
```

也可以右键 `index.html`，选择用 Edge 或 Chrome 打开。

## 每个文件是做什么的

- `index.html`：网页的主要内容，包括首页、关于我、内容入口、播客、科研背景、数字分身和联系区。
- `styles.css`：网页的视觉样式，包括颜色、排版、按钮、卡片和手机端适配。
- `script.js`：网页的小功能，包括中文 / EN 切换、主题卡片文字切换、数字分身按钮提示。
- `assets/avatar.png`：数字分身头像。
- `assets/wechat-qr.png`：微信二维码。
- `assets/favicon.svg`：浏览器标签页的小图标。
- `README.md`：项目说明，也就是你现在看到的这个文件。

## 如果要上线到 Vercel

大概步骤是：

1. 注册或登录 Vercel。
2. 把这个项目上传到 GitHub，或用 Vercel 导入项目。
3. 在 Vercel 里选择这个项目。
4. 点击 Deploy。
5. 等待部署完成，得到一个可以分享的网址。

因为这是静态网页，通常不需要复杂设置。如果 Vercel 询问框架类型，可以选择 `Other`。Build Command 和 Output Directory 一般可以留空。

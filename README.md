# YAO / 瑶瑶个人主页

这是 YAO / 瑶瑶的个人品牌主页项目，用来展示她的生物医药背景、澳洲工作生活、内容创作、播客、数字分身入口、访客留言和合作联系方式。

当前版本是 V5。项目使用普通 HTML、CSS 和 JavaScript，没有复杂框架，可以直接本地打开，也适合之后部署到 Vercel 或 Netlify。

## V5 新增了什么

V5 增加了“留言给瑶瑶 / Leave a note for YAO”互动区。访问者可以留下：

- 合作想法
- 想听的话题
- 对 AI、投资学习、生物医药、澳洲生活、女性成长的好奇
- 简单打招呼

当前提交方式使用最简单稳定的 `mailto`：点击提交后，会自动整理留言内容，并尝试打开访问者自己的邮箱客户端，发送到 `yaoyaosocial01@gmail.com`。

这个版本不需要后端、不接数据库、不保存用户信息到本地。以后如果想升级，可以接 Formspree、Tally、Notion 表单、Google Forms，或者自己的数据库。

## 本地怎么打开

直接双击打开这个文件：

```text
D:\个人主页\index.html
```

也可以右键 `index.html`，选择用 Edge 或 Chrome 打开。

## 每个文件是做什么的

- `index.html`：网页的主要内容，包括首页、关于我、内容入口、播客、科研背景、数字分身、留言区和联系区。
- `styles.css`：网页的视觉样式，包括颜色、排版、按钮、卡片、留言表单和手机端适配。
- `script.js`：网页的小功能，包括中文 / EN 切换、主题卡片文字切换、数字分身预设问答、留言表单整理邮件。
- `assets/avatar.png`：数字分身头像。
- `assets/wechat-qr.png`：微信二维码。
- `assets/favicon.svg`：浏览器标签页的小图标。
- `README.md`：项目说明，也就是你现在看到的这个文件。

## 如果要上线到 Vercel

大概步骤是：

1. 注册或登录 Vercel。
2. 把这个项目上传到 GitHub，或用 Vercel 导入项目。
3. 在 Vercel 里选择这个项目。
4. 点击 Deploy。
5. 等待部署完成，得到一个可以分享的网址。

因为这是静态网页，通常不需要复杂设置。如果 Vercel 询问框架类型，可以选择 `Other`。Build Command 和 Output Directory 一般可以留空。

## 上线前检查

- 所有链接可以打开。
- 中文 / EN 切换正常。
- 手机端显示正常。
- `avatar.png` 头像显示正常。
- 留言表单可以打开邮箱客户端。
- 联系邮箱 `yaoyaosocial01@gmail.com` 正确。
- 微信号 `y13534369137` 正确。
- 页面里没有 `C:\` 或 `D:\` 这样的本地绝对路径。
- 页面里没有不想公开的私人信息。
